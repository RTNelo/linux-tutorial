# 关于本书

这是一本 Linux 的简明入门教程，基于东北大学软件学院石凯老师的讲义，由同学整理并扩充，与东北大学软件学院 Linux 操作系统课程同步。

## 技能要求

因为该课程面向的是软件工程大三年级的学生，所以我们默认本书的读者已经具备一定的 C 语言编程基础，同时具备一定的英语能力。
